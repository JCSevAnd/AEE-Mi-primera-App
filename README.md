# AEE-Mi-primera-App
Desarrollador: Juan Calixto.
Descripción: Aplicación Android con 5 Activities utilizando Kotlin y View Binding (inflate).

## Funcionalidades
1. Login con paso de datos.
2. Inicio con navegación y recepción de resultados.
3. Formulario de contacto que devuelve datos.
4. Apartado sobre nosotros.
5. Localización mediante simulación de mapa.
